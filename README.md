# Laundry-wallah
A simple interface of laundry wallah html web page

🧺 Laundry Wallah / Laundry Mart

Overview

Laundry Wallah / Laundry Mart is a simple HTML-based landing page for an online laundry service. It showcases the services offered, pricing details, and a booking form for customers. The project is built using semantic HTML5 elements to ensure a clean structure and improved accessibility.

Features

- Responsive HTML5 structure
- Semantic HTML tags
- Services section
- Price list in a table
- Booking form with input validation
- Clickable email contact link
- Image with caption

Technologies Used

- HTML5

How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click "index.html" or open it in any modern web browser.
